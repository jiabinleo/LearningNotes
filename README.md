```
如何判断this 指向 就需要找到这个函数的直接调用位置 找位置之后再确定this指向
1.new 调用 直接指向new 创建的新对象
2.上下文调用 直接指定上下文
3.call .apply 指向指定的 绑定对象
4.全局函数下 指向全局
es6的箭头函数 指向外层函数的this
```
